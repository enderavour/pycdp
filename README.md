# pycdp - Small wrapper over Google CDP Protocol in Python
### Implements:
- Page screenshots 
- Navigation to URL
- Sendins keys to pages
- Executing JS scripts

### To install dependencies:
```cmd
pip install -r requirements.txt
```

### To run an example:
```cmd
python example.py
```
